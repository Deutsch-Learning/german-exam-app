## 📘 German Learning App

A simple and interactive application designed to help users learn and practice the German language effectively. Whether you're a beginner or looking to improve your skills, this app provides structured lessons, exercises, and real-life practice.

---

## 🚀 Features

* 📖 **Structured Lessons** – Learn German step by step (A1 to advanced levels)
* 🧠 **Vocabulary Practice** – Flashcards and quizzes to improve word retention
* 🗣️ **Pronunciation Training** – Practice speaking with audio support
* ✍️ **Grammar Exercises** – Understand German grammar with interactive exercises
* 🎯 **Progress Tracking** – Monitor your learning journey
* 📱 **User-Friendly Interface** – Simple and intuitive design

---

## 🛠️ Technologies Used

* Frontend: (React, HTML/CSS/JS)
* Backend: ( Node.js)
* Database: ( PostgreSQL)

---

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/Yong-Justice/german-exam-app.git
```

2. Navigate into the project folder:

```bash
cd german-exam-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```

---

## 🧪 Usage

* Sign up or log in
* Choose your German level
* Start lessons and complete exercises
* Track your progress over time
